What is Visual Code and who it is for

VisualCode is a tool for Processing IDE. Taking your code as input, it generates a text file meant to be replayed in Processing. 
This text file is an enhanced version of your sketch in which, through a switch from the keyboard, you can view both the original sketch and it's lines of code, running on real time.
Understanding in detail how runs the code can facilitate debugging, organization and optimization of the code itself. 
Visual Code can also be a useful support for educational activities related to programming in Processing environment.


Instructions

-Open the program and using the keyboard command Ctrl + V paste the segments of the code following the instructions. 
NB: Before you paste the code, it should be formatted in your Processing text editor: drop-down menu, click on Edit, then Auto Format (shortcut Ctrl + T).
-Open the text file you just created "code.txt", copy and paste the contents into a new sketch in Processing, then run.
Controls: 
Use the buttons 'S' (show) and 'H' (hide) to show or hide the code of the sketch. 
Use buttons '+' and '-' to zoom in or out the code. 
Move the code by dragging it with the mouse, press '0' to bring it back to its initial position.


How to read the results

Initially, the code includes everything in the size of your sketch. 
The lines of code running in every moment are highlighted by a green background more or less intense, depending on their frequency of execution. 
The lines of code that are not running on the other hand, are white and invisible.

Cos'è Visual Code e a chi è destinato

Visual Code è un tool per Processing IDE che, prendendo in input il tuo codice, genera un file di testo pensato per essere rieseguito in Processing. 
Questo file di testo costituisce una versione aumentata del tuo sketch, in cui tramite uno switch da tastiera è possibile visualizzare sia lo sketch originale, sia le righe di codice che lo compongono, in esecuzione momento per momento.
Capire nel dettaglio come si comporta il codice può facilitare le operazioni di debug, organizzazione e ottimizzazione del codice stesso. 
Visual Code può essere inoltre un utile supporto per attività didattiche legate alla programmazione in ambiente Processing.


Istruzioni

-Aprire il programma e incollare con il comando da tastiera Ctrl+V le parti di codice seguendo le istruzioni. 
N.B.: Prima di incollare il codice, questo va formattato nel text editor di Processing: dal menù a tendina fare click su Edit, quindi Auto Format (da tastiera Ctrl+T).
-Aprire il file di testo appena creato "code.txt", copiare e incollare il contenuto in un nuovo sketch di Processing, quindi eseguire.
Comandi: 
Usa i tasti 'S' (show) e 'H' (hide) per mostrare o nascondere il codice dello sketch. 
Usa i tasti '+' e '-' per ingrandire o rimpicciolire il codice. 
Sposta il codice trascinandolo con il mouse, premi '0' per riportarlo nella posizione iniziale.


Come leggere i risultati

Inizialmente il codice rientra tutto nel size del tuo sketch. 
Le righe di codice in esecuzione in ogni momento sono evidenziate da uno sfondo verde più o meno intenso a seconda della loro frequenza di esecuzione. 
Le righe di codice non in esecuzione invece, sono bianche e invisibili.
